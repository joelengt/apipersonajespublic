#Detalle


Este es un API REST publico, permite usar toda la data de personajes dentro de nuestra base de datos. Los permisos del API permiten peticiciones desde cualquier sitio web en los metodos: GET, POST, PUT, DELETE. El proposito de este contenido y el uso, es con fines educativos, toda imagen y dato enviado se almacena en nuestra base de datos. Recordar, dado a que al tener los permisos de crear, actualizar y borrar personajes, los contenidos inapropiados seran eliminados.



#Uso


Para usar la data, solo debes hacer un petición (desde un formulario o AJAX) en los siguientes:


  • Obtener todos los personajes

    metodo GET:


    http://apipersonajes.herokuapp.com/api/personajes

  • Obtener un personaje por id

    metodo GET:


    http://apipersonajes.herokuapp.com/api/personaje/:id

    Ejemplo:


    http://apipersonajes.herokuapp.com/api/personaje/56ee4796d20081181d87e824

  • Crear nuevo personaje

    metodo POST:


    Para crear un nuevo personaje, se require llenar los parametros: name, occupation, group, age, photo (Todo tipo String, age es Number)

    Solo necesita, pasar parametros en la petición (como con AJAX)

    http://apipersonajes.herokuapp.com/api/personaje

    También con Socket.io:

    Requerir en el html jquery, socket.io

    Colocar el js en emit, on mediante el canal: 'new personaje'


    Ejemplo:


    var socket = io()
    $('form').submit(function () {
    socket.emit('new personaje', {
    id: '',
    name: $('#personaje_name').val(),
    occupation: $('#personaje_occupation').val(),
    group: $('#personaje_group').val(),
    age: $('#personaje_age').val(),
    photo: $('#personaje_photo').val()
    })
    return false
    })

    socket.on('new personaje', function (content) {
    $('body').prepend(content.name)
    })


    El id del personaje es generado automaticamente por la base de datos. El personaje creado debe llegar como estado '200 OK', donde muestra los datos del elemento creado.


  • Borrar un personaje

    metodo POST


    *La mayoria de navegadores aun no soportan los metodos Delete desde el navegador, asi que se usa algo parecido que cumple lo mismo.


    http://apipersonajes.herokuapp.com/api/personaje/:id?_method=delete

    Ejemplo:


    http://apipersonajes.herokuapp.com/api/personaje/4423jkfivn43dwelk324?_method=delete

    La efectividad del Proceso DELETE, te devolvera un status '200 OK'


  • Actualizar un personaje

    metodo POST


    *La mayoria de navegadores aun no soportan los metodos Put desde el navegador, asi que se usa algo parecido que cumple lo mismo.


    http://apipersonajes.herokuapp.com/api/personaje/:id?_method=put

    Ejemplo:


    http://apipersonajes.herokuapp.com/api/personaje/4423jkfivn43dwelk324?_method=put

    La efectividad del Proceso PUT, te devolvera un status '200 OK'



#Permisos


El uso del API, estan habilitados a todos los sitios web, para usar los metodos: GET, POST, DELETE, PUT, a los datos de la aplicación, todo con fines educativos.
No se require ningun registro, ni cosa alguna. Para usar la data solo se necesita hacer una peticion GET o POST, ya sea desde un formulario o AJAX e incluso socket.io, tal cual en la guia de uso.
